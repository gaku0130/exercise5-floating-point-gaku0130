2. Geometric Meaning of Matrix Condition Number

Recall from the handout that condition numbers can be defined with respect to different matrix norms. In this exercise, always use the 2-norm condition number:

\[ \operatorname{cond}_2(A) = \|A\|_2\,\|A^{-1}\|_2 = \frac{\sigma_{\max}(A)}{\sigma_{\min}(A)}, \]

where \(\sigma_{\max}(A)\) and \(\sigma_{\min}(A)\) are the largest and smallest singular values of \(A\). In NumPy, LA.cond(A) uses the 2-norm by default.

For this exercise, focus on the following \(2\times 2\) symmetric positive definite matrix:

A_bad_example = np.array([
    [2.0, 1.9],
    [1.9, 2.0],
])

print("eigenvalues:", LA.eigvalsh(A_bad_example))
print("matrix condition number:", LA.cond(A_bad_example))

The idea is to prescribe the singular values directly. If

\[ A = Q_1 \Sigma Q_2^T \]
